# 基于STM32的基于STM32的无线语音控制小车

## 一、项目大概

### 1、软件组成

STM32主程序、声音识别程序、AndroidAPP程序

### 2、硬件组成

STM32 微处理器、HC-06蓝牙模块、LD3320声音识别模块、0.96寸OLED显示器、HC-SR04超声波传感器、L298N 电机驱动

### 3、开发语言

C语言、Java
